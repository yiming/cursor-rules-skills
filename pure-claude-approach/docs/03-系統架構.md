# 系統架構

## 目錄結構

```
pure-claude-approach/
├── CLAUDE.md                              ← 飯店經理：政策 + 團隊分工
│
├── .claude/                               ← Claude 原生配置
│   ├── settings.json                      ← 信任邊界（強制權限）
│   ├── skills/
│   │   ├── process-order/                 ← /process-order 指令
│   │   │   ├── SKILL.md                   ← 流程定義（frontmatter + 步驟）
│   │   │   └── templates/
│   │   │       ├── confirmation.md        ← 確認單模板
│   │   │       └── task-list.md           ← 任務清單模板
│   │   └── check-availability/            ← /check-availability 指令
│   │       └── SKILL.md                   ← 動態上下文注入示例
│   └── agents/                            ← 子代理定義
│       ├── order-clerk.md                 ← 訂單處理員
│       └── auditor.md                     ← 稽核員
│
├── master-data/                           ← 主資料（唯一價格來源）
│   ├── CLAUDE.md                          ← 主資料保護規則
│   ├── room-types.yaml                    ← 房型、價格、加購選項
│   └── room-list.yaml                     ← 房號清單
│
├── orders/                                ← 訂單
│   ├── CLAUDE.md                          ← 訂單結構契約
│   ├── {id}.req.md                        ← 客戶需求（唯讀）
│   ├── {id}-確認單.md                      ← AI 產出：確認單
│   └── {id}-任務.md                        ← AI 產出：任務清單
│
└── docs/                                  ← 設計文件
```

## 處理流程

```
使用者：/process-order BR250315-001
     │
     ├── ① 觸發 process-order skill
     │      └── 讀取 SKILL.md + templates/
     │
     ├── ② 委派 order-clerk（sonnet）
     │      ├── 讀取 orders/BR250315-001.req.md
     │      ├── 載入 master-data/room-types.yaml
     │      ├── 載入 master-data/room-list.yaml
     │      ├── 計算房價、匹配房型、篩選房號
     │      ├── 品質檢查
     │      ├── 寫入 orders/BR250315-001-確認單.md
     │      └── 寫入 orders/BR250315-001-任務.md
     │
     └── ③ 委派 auditor（haiku）
            ├── 讀取確認單
            ├── 比對 master-data 金額
            ├── 驗證房號存在
            ├── 確認容量足夠
            └── 回報：✓ 通過 / ✗ 不通過
```

## 與舊版架構的對比

| 面向 | 舊版（規則模式） | 新版（團隊模式） |
|------|----------------|----------------|
| 頂層 | CLAUDE.md 規則手冊 | CLAUDE.md 團隊政策 |
| 觸發 | 派發表 → skills/ | `/指令` → .claude/skills/ |
| 子目錄規則 | AGENTS.md | CLAUDE.md（正式機制） |
| 流程 | skills/*.md（鬆散） | .claude/skills/*/SKILL.md（正式） |
| 模板 | 嵌在 skill 內 | 獨立 templates/ 目錄 |
| 驗證 | 無 | auditor 子代理 |
| 權限 | 文字提醒 | settings.json 強制 |
| 模型 | 單一 | sonnet + haiku |
