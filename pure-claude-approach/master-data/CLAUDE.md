# 主資料管理規則

本目錄存放飯店營運的基礎資料，是所有訂單計算的唯一價格來源。

## 檔案清單

| 檔案 | 用途 |
|------|------|
| `room-types.yaml` | 房型定義：名稱、容量、每晚價格、加購選項 |
| `room-list.yaml` | 房號清單：樓層、房號、對應房型 |
| `menu.yaml` | 餐飲品項：正餐/飲料/點心價格、低消規則（config） |
| `venues.yaml` | 場地定義：名稱、容量、場次價格、宴會餐標、加購選項 |

## 規則

- 主資料是**唯一的價格與房號來源**，所有產出的金額與房號必須可追溯到這裡。
- 修改主資料前應確認不影響已產出的訂單。
- 新增房型或房號時，確保 ID 不與既有資料重複。
