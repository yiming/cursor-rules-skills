# 查詢空房狀態

## 使用方式

```
@hotel-check-availability 查 2025-04-20 到 2025-04-22 的空房
```

## 步驟

### Step 1：載入房間資料

讀取以下兩個檔案：
- `master-data/room-types.yaml`：取得各房型名稱、容量、每晚價格
- `master-data/room-list.yaml`：取得所有房號清單（room_type, floor, number）

### Step 2：掃描已預訂房號

讀取 `orders/` 目錄下所有 `*-確認單.md`，從每張確認單的「建議房號」欄位提取已預訂的房號。

篩選條件：確認單的入住日期與退房日期與查詢區間有重疊（任何一晚重疊都算已預訂）。

若無法從確認單判斷日期，保守處理：視為已預訂。

### Step 3：計算可用房號

從 room-list.yaml 的完整房號清單中，排除 Step 2 找到的已預訂房號。

### Step 4：以表格呈現結果

按房型分組輸出：

```
查詢期間：{check_in} ～ {check_out}（共 {nights} 晚）

## 可用房間

### 標準雙人房（STD-D）｜容量：2 人｜每晚 3,200 元

| 房號 | 樓層 | 狀態 |
|------|------|------|
| 301 | 3F | ✓ 可用 |
| 302 | 3F | ✓ 可用 |
| 303 | 3F | ✗ 已預訂 |
...

可用：{n} 間 / 共 {total} 間

### 豪華雙床房（DEL-T）...
```

## 注意事項

- 若所有房型均無空房，明確告知並建議其他日期
- 若查詢期間跨月，正常處理
- 結果僅供參考，實際預訂以確認單為準
